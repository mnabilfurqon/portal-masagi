# How to Run Website MASAGI Internal Attendance

### Prerequisite 
- install yarn
```bash
npm install --global yarn
```
- make sure you can run the backend side first
```bash
https://github.com/Mitra-Solusi-Aktual-Integrasi/internal-attendance-backend
```

### Step by step
```bash
1. $ git clone https://github.com/Mitra-Solusi-Aktual-Integrasi/internal-attendance-frontend.git
2. $ cd internal-attendance-frontend
3. open folder in code editor
4. $ git checkout development
5. $ yarn
6. $ yarn run dev
7. go to http://localhost:3000
```
